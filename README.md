# Projeto Promobit-Filmes

### Funcionalidades 
O projeto entrega uma lista com os filmes mais populares da atualidade.

* Filtrar por categorias 
* ver todos os detalhes do filme ao clicar no filme de sua preferencia
* Paginação por ordem 

## Tecnologias

* ReactJS.
* Styled Components.
* Axios.
* Routes
* Context
